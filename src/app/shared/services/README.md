# Servicios Compartidos (Shared Services)

Este directorio contiene servicios de Angular que son reutilizables y pueden ser consumidos por diferentes componentes a lo largo de la aplicación.

A diferencia de los servicios en `core`, estos suelen tener una funcionalidad más específica y no son necesariamente singleton para toda la aplicación, aunque a menudo lo son. Son compartidos entre `features`, pero no son tan fundamentales como para estar en el `core`.
