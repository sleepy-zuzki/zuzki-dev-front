# Modales (Modals)

Este directorio contiene componentes que están diseñados para ser presentados como ventanas modales.

Estos componentes suelen ser invocados dinámicamente por un servicio y se muestran por encima del contenido principal de la página.

## Componentes

-   **`project-info-modal.component`**: Un modal para mostrar información detallada de un proyecto.
