# Modelos (Models)

Este directorio contiene clases de TypeScript que actúan como modelos de datos para la aplicación.

A diferencia de las `interfaces`, las clases pueden tener constructores, métodos y propiedades. Se utilizan a menudo para encapsular datos y comportamiento relacionado, creando modelos "ricos" que pueden, por ejemplo, formatear datos o realizar cálculos simples sobre sí mismos.

Estos modelos suelen representar los datos de una forma que es conveniente para la capa de presentación (UI).
