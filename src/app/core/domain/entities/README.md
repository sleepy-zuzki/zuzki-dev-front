# Domain Entities

Esta carpeta contiene las **entidades del dominio**, que son los objetos fundamentales del negocio.

## Estructura
- Las entidades representan conceptos centrales del dominio
- Contienen la lógica de negocio pura
- Son independientes de cualquier framework o tecnología externa
- Definen las reglas de negocio y validaciones
