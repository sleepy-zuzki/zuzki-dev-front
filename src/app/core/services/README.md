# Servicios del Core

Este directorio contiene servicios de Angular que son fundamentales y tienen un alcance global en toda la aplicación. A diferencia de los servicios de dominio, estos pueden estar acoplados al framework (Angular) y se encargan de lógica de la aplicación que no es estrictamente lógica de negocio.

## Ejemplos de Servicios

-   **`SeoService`**: Servicio para gestionar las meta-etiquetas SEO (título, descripción) de las páginas.
-   Servicios de gestión de estado global.
-   Servicios de logging o manejo de errores.
