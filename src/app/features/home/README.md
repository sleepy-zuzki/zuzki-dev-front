# Feature: Home

Este directorio contiene todos los componentes y la lógica relacionados con la sección "Home" de la página.

La página de inicio está construida ensamblando varios sub-componentes, cada uno representando una sección de la página.

## Subdirectorios / Componentes

-   **/hero**: La sección principal o de "héroe" al inicio de la página.
-   **/about**: La sección que contiene información sobre mí.
-   **/services**: Muestra los servicios que ofrezco.
-   **/projects**: Presenta una vista previa de los proyectos destacados.
-   **/contact**: La sección con información de contacto.
