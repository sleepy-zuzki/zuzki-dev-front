# Feature: Admin

Este directorio contiene los componentes y la lógica para el panel de administración del portafolio.

Desde aquí se gestiona el contenido dinámico de la aplicación, como los proyectos.

## Subdirectorios / Componentes

-   **/projects**: Componente para la gestión (CRUD) de los proyectos del portafolio.
-   **/technologies**: Componente para la gestión de tecnologías.
-   **/gallery**: Componente para la gestión global de imágenes.
