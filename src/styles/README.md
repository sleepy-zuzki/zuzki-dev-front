# Estilos Globales (Styles)

Este directorio contiene archivos de estilos CSS que se aplican globalmente a toda la aplicación.

Estos archivos son importados en el archivo principal `styles.css` o configurados en `angular.json` para ser incluidos en la compilación.

## Archivos

-   **`colors.css`**: Define las variables de color principales utilizadas en la aplicación, siguiendo la paleta de Tailwind CSS.
-   **`cards.css`**: Estilos reutilizables para los componentes de tipo "tarjeta".
