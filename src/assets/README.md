# Assets

Este directorio es gestionado por Angular y contiene recursos estáticos que se procesan e incluyen en la compilación de la aplicación.

A diferencia de la carpeta `/public`, los archivos aquí son importados directamente en los componentes, plantillas o estilos (por ejemplo, una imagen de fondo en un CSS).

## Contenido Típico

-   Imágenes (JPG, PNG, SVG) que se usan dentro de los componentes.
-   Archivos de datos JSON locales.
-   Iconos personalizados.

La configuración de qué carpetas se tratan como `assets` se encuentra en `angular.json`.
