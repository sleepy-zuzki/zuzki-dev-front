# Herramientas (Tools)

Este directorio contiene scripts y herramientas personalizadas que asisten en el proceso de desarrollo y construcción del proyecto.

Estos scripts no forman parte de la aplicación en sí, sino que son utilidades para el desarrollador.

## Scripts

-   **`copy-files.mjs`**: Un script de Node.js que se ejecuta después del proceso de `build` para copiar archivos necesarios (como `_redirects`) a la carpeta de distribución final.
